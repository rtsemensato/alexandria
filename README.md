# Alexandria
Software de gerenciamento da biblioteca da Fafiman

## Configuração

* Java 1.8
* Maven 3.6.0
* IntelliJ IDEA

### Maven

Criação do projeto:
```
mvn archetype:generate  -DgroupId=br.fafiman.alexandria \
                        -DartifactId=alexandria \
                        -DarchetypeArtifactId=maven-archetype-webapp \
                        -DinteractiveMode=false
```

Build:
```
mvn clean install
```
